# AI Talent Lab 문의하기 Agent PoC 분석 리포트

## 개요

본 문서는 AI Talent Lab 문의하기 페이지에 Agent를 적용하기 위한 PoC 분석 결과를 담고 있습니다.

**분석 대상**: 84건의 문의, 90건의 댓글, 14건의 첨부 파일

**분석 일자**: 2026-02-15

---

## 1. 답변이 가능한 질문과 그렇지 않은 질문

### 1.1 답변 가능성 분포

| 분류 | 건수 | 비율 | Agent 적용 가능 여부 |
|------|------|------|---------------------|
| **답변 가능** | 70건 | 83.3% | ✅ Agent가 답변 가능 (38건) 또는 조건부 가능 (15건) |
| **답변 불가능** | 9건 | 10.7% | ❌ 운영자 직접 처리 필요 (24건) |
| **조건부 답변 가능** | 5건 | 6.0% | ⚠️ 추가 정보에 따라 판단 |

### 1.2 답변 가능한 질문 유형

#### ✅ **학습/과제 관련 (40건, 47.6%)**

**특징**: 일반적인 학습 과정, 과제 제출, 강의 관련 질문

**예시**:
- "최종 과제 기획 설계 문서와 소스코드는 여러 번 제출 가능한가요?"
- "파이썬 문법 강의를 듣고 싶은데 어디서 들을 수 있나요?"
- "강의 영상이 재생되지 않습니다"

**Agent 답변 가능 이유**:
- FAQ 데이터베이스 또는 가이드 문서 기반 답변 가능
- 과거 유사 질문의 답변 패턴 학습 가능
- 일관된 규칙 적용 가능 (예: 제출 규정, 수강 안내 등)

**필요한 Context**:
- 강의 커리큘럼 정보
- 과제 제출 규정 및 가이드
- 학습 자료 링크
- 과거 문의 답변 히스토리

---

#### ⚠️ **기술지원/에러 관련**

**특징**: 코드 로직 에러, 플랫폼 사용법 문의

**참고**: AI Talent Lab은 이미 구성된 환경(클라우드 IDE/Jupyter)을 제공하므로, 개인 환경 설정 문제(API 키, 라이브러리 설치)는 발생하지 않음

**예시**:
- "코드 실행 시 에러가 발생합니다" (사용자 코드 로직 문제)
- "플랫폼 API 사용 방법" (플랫폼 기능 사용법)
- "강의 영상이 재생되지 않습니다"

**Agent 답변 가능 조건**:
- ✅ **가능**: 사용자 코드 로직 에러, 플랫폼 사용법
  - 과거 유사 에러 사례 검색 (RAG)
  - 코드 디버깅 제안
  - 플랫폼 사용 가이드 제공
- ❌ **불가능**: 플랫폼 시스템 자체 문제
  - 백엔드 서버 에러 (console 접근 불가 등)
  - 시스템 점검 필요

**필요한 Context**:
- RAG: 과거 코드 에러 해결 사례
- FAQ: 플랫폼 사용 가이드
- FAQ: 영상 재생 트러블슈팅

---

### 1.3 답변 불가능한 질문 유형

#### ❌ **계정/권한 관련**

**특징**: 개인 계정 확인 및 시스템 조치 필요

**예시**:
- "인증시험 버튼이 비활성화 되어있습니다"
- "AI Literacy 인증 시작하기 버튼 활성화가 안됩니다"
- "과제 제출 권한이 없습니다"

---

#### ❌ **플랫폼 시스템 에러**

**특징**: 플랫폼 백엔드 문제로 긴급 조치 필요

**예시**:
- "console 접근/python script 실행이 안됩니다"
- "파일 업로드가 안됩니다"

**Agent 답변 불가능 이유**:
- 백엔드 시스템 점검 및 로그 확인 필요
- 긴급 조치 필요 (우선순위: 높음)

**Agent 답변 불가능 이유**:
- **개인 계정 정보 확인 필요**: 사용자별 권한, 그룹, 인증 대상 여부 확인
- **시스템 직접 조치 필요**: 데이터베이스 레벨에서 권한 변경
- **보안 이슈**: 개인 정보 보호를 위해 운영자만 접근 가능

**운영자 답변 패턴**:
- 대부분 "안녕하세요! 조치해드렸습니다" 형태의 단순 조치 완료 답변
- Agent는 이런 조치를 직접 수행할 수 없음

**Agent의 역할**:
- ✅ 일반적인 계정 관련 안내는 가능
  - 예: "인증 시험 버튼은 인증 대상자에게만 활성화됩니다. 인증 대상자 안내를 받지 않으셨다면 비활성화 상태가 정상입니다."
- ❌ 개인 계정 조치는 불가능
  - 운영자 에스컬레이션 필요

**필요한 Context**:
- 사용자 계정 정보 (권한, 그룹, 인증 대상 여부) - **접근 권한 문제로 제한적**
- 인증 시험 일정 및 대상자 정보
- 사용자별 활성화된 기능 목록

---

#### ⚠️ **건의/개선 요청 (1건, 1.2%)**

**특징**: 서비스 개선, 새로운 기능 요청

**예시**:
- "대시보드에 학습 진도율을 표시해주세요"
- "모바일 앱도 제공해주세요"

**Agent 답변 가능 조건**:
- ✅ **가능**: 기존 로드맵, 개선 이력 공유
- ❌ **불가능**: 실제 개선 여부 결정, 우선순위 판단 (운영팀 판단 필요)

**필요한 Context**:
- 현재 서비스 기능 명세
- 개선 요청 이력
- 로드맵 정보 (공개 가능한 범위)

## 3. 과거 응답을 바탕으로 일관성 있게 참고해서 답변하게 만들 수 있는지

### 3.1 결론: ✅ 가능합니다

**조건**:
1. 과거 운영자 답변을 학습 데이터로 활용
2. 답변 패턴 및 톤앤매너 분석
3. 유사 질문 매칭 시스템 구축

---

### 3.2 운영자 답변 패턴 분석

현재 데이터셋에서 **90건의 운영자 댓글** (user_id: 2, 7, 61) 분석 결과:

#### 패턴 1: 단순 조치 완료 (가장 많음)

**예시**:
- "안녕하세요! 조치해드렸습니다."
- "안녕하세요 조치 완료했습니다."

**특징**:
- 50자 미만의 짧은 답변
- 계정/권한 문제에 주로 사용
- **Agent 적용 불가**: 실제 조치를 수행할 수 없음

**Agent 대응 방안**:
```
안녕하세요.

[문제 상황 요약]

해당 문제는 시스템 관리자의 직접 조치가 필요한 사항입니다.
운영팀에 문의를 전달하였으며, 빠른 시일 내에 조치 후 답변드리겠습니다.

감사합니다.
```

---

#### 패턴 2: 가이드 제공 (학습/과제 관련)

**예시**:
```
안녕하세요.
AI Literacy 인증 시작하기 버튼은 인증 시험 응시자 대상으로 기간 내 활성화됩니다.
인증 대상자 안내를 받지 않으셨다면 비활성화 상태가 정상 상태입니다.
```

**특징**:
- 명확한 설명 제공
- 정책/규정 안내
- **Agent 적용 가능**: 일관된 규칙 기반 답변

**Agent 대응 방안**:
- FAQ 데이터베이스에 저장
- 키워드 매칭으로 자동 답변
- 과거 답변을 템플릿으로 활용

---

#### 패턴 3: 상세 설명 (100자 이상)

**예시**:
```
안녕하세요.
최종과제의 기획 설계 문서와 소스코드는 여러번 제출 가능하십니다.
단, 마감 기한 이내에 제출하신 최종 버전으로 평가됩니다.
```

**특징**:
- 구체적인 정보 제공
- 조건/제약사항 명시
- **Agent 적용 가능**: 과거 답변 학습하여 유사 형식으로 답변

### 3.3 일관성 유지 전략

#### ✅ 1. RAG (Retrieval-Augmented Generation) 방식

**방법**:
1. 과거 운영자 답변을 벡터 DB에 저장
2. 새로운 질문이 들어오면 유사한 과거 질문 검색
3. 검색된 답변을 참고하여 새로운 답변 생성

**장점**:
- 과거 답변 스타일 자동 학습
- 일관된 톤앤매너 유지
- 새로운 답변도 과거 패턴 따름

**구현 예시**:
```python
# 유사 질문 검색
similar_inquiries = search_similar(new_question, past_inquiries)

# 과거 답변 추출
past_responses = [inq['admin_response'] for inq in similar_inquiries]

# 프롬프트 구성
prompt = f"""
과거 유사 질문에 대한 운영자 답변:
{past_responses}

새로운 질문: {new_question}

위 과거 답변의 톤앤매너와 스타일을 참고하여 답변을 작성하세요.
"""
```

#### ✅ 2. Few-Shot Learning

**방법**:
- Agent 프롬프트에 좋은 답변 예시 5~10개 포함
- 각 질문 유형별 대표 예시 선정

**예시**:
```
당신은 AI Talent Lab 고객 지원 Agent입니다.
다음은 운영자의 실제 답변 예시입니다:

[예시 1 - 학습 관련]
Q: 최종 과제는 여러 번 제출 가능한가요?
A: 안녕하세요. 최종과제의 기획 설계 문서와 소스코드는 여러번 제출 가능하십니다.

[예시 2 - 권한 관련]
Q: 인증 버튼이 비활성화되어 있습니다.
A: 안녕하세요. AI Literacy 인증 시작하기 버튼은 인증 시험 응시자 대상으로 기간 내 활성화됩니다.

위 스타일을 참고하여 답변하세요.
```

#### ✅ 3. 답변 품질 검증

**방법**:
1. Agent가 답변 생성
2. **자가 검증 단계**: Agent가 스스로 답변 품질 평가
   - 과거 답변과 유사한가?
   - 정확한 정보를 제공하는가?
   - 추가 정보가 필요한가?
3. 신뢰도 점수 부여
4. **낮은 신뢰도 시 운영자 에스컬레이션**

**예시**:
```python
confidence_score = agent.evaluate_response(generated_response, past_responses)

if confidence_score < 0.7:
    # 운영자 검토 필요
    flag_for_review(inquiry, generated_response)
else:
    # 자동 답변 게시
    post_response(generated_response)
```

### 3.4 다국어 대응 전략

**현황**:
